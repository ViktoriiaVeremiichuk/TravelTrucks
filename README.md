# TravelTrucks

TravelTrucks — це вебзастосунок для пошуку та бронювання кемперів. Користувач
може переглядати каталог доступних кемперів, застосовувати фільтри, переглядати
детальну інформацію про кожен кемпер, ознайомлюватися з відгуками та надсилати
заявку на бронювання.

## Основні можливості

- Домашня сторінка з банером та закликом до дії.
- Каталог кемперів із завантаженням даних із бекенду.
- Фільтрація кемперів за:
  - локацією;
  - типом кузова;
  - типом двигуна;
  - типом трансмісії.
- Пагінація у форматі **Load More**.
- Перехід на сторінку деталей кемпера в новій вкладці.
- Галерея фотографій кемпера.
- Відображення рейтингу та відгуків користувачів.
- Форма бронювання з відправкою даних на сервер.
- Нотифікація про успішне бронювання.

## Встановлення та запуск

### 1. Клонувати репозиторій

bash git clone https://github.com/your-username/travel-trucks.git

### 2. Перейти в папку проєкту

bash cd travel-trucks

### 3. Встановити залежності

bash npm install

### 4. Запустити проєкт у режимі розробки

bash npm run dev

Після запуску застосунок буде доступний за адресою: http://localhost:3000

## Структура застосунку

├── Home
│   └── Банер із переходом до каталогу
│
├── Catalog
│   ├── список кемперів
│   ├── фільтри
│   └── пагінація Load More
│
└── Camper Details
    ├── інформація про кемпер
    ├── галерея
    ├── відгуки
    └── форма бронювання

## Особливості реалізації

- Фільтрація виконується на стороні бекенду через query-параметри.
- Підтримується довантаження карток без втрати активних фільтрів.
- Детальна сторінка відкривається в новій вкладці.
- Стан застосунку керується за допомогою Redux Toolkit.
- Збереження необхідних даних реалізовано через Redux Persist.
